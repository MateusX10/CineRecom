Descrição do projeto:

A CineRecom é um Sistema de Recomendação(SR) de Filmes e Séries que tem como objetivo fazer recomendações de obras ao usuário de forma precisa de acordo com seus gostos pessoais e interações com o sistema.É possível deixar review, comentar, dar nota e entre outros no sistema.A medida que o comportamento do usuário perante o sistema muda, o comportamento do sistema perante o usuário também muda, com o sistema recomendando as obras que se encaixam nos gostos presentes do usuário e também futuros.
